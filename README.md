# Arch Linux + niri 安装与配置记录

本仓库用于记录 Arch Linux 的安装方式、桌面环境配置，以及日常使用中的相关配置技巧。内容会以实用为主，方便在重新安装系统、迁移环境或排查问题时快速查阅。

## 内容范围

- Arch Linux 安装流程与注意事项
- 分区、引导、网络、用户、权限等基础配置
- niri 窗口管理器相关配置
- 常用软件、开发工具与桌面组件配置
- 输入法、字体、主题、终端、Shell 等使用体验优化
- Obsidian 等个人知识管理工具的配置与插件记录
- 常见问题排查与踩坑记录

## 适用场景

这个仓库主要面向个人环境维护，也可以作为 Arch Linux 安装和配置的参考资料。由于硬件、网络环境和个人习惯不同，部分配置可能需要根据实际情况调整。

Arch Linux 的软件包、内核和桌面生态更新较快，文档中的命令和配置会随使用过程持续调整。实际操作前建议结合官方文档确认最新信息。
